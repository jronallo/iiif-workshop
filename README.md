# IIIF Workshop

Materials for a workshop covering:
- Image API
- more to come....

## Development

Requires NodeJS.

- Clone the repository
- `cd iiif-workshop/workshop`
- `npm i gitbook-cli -g`
- `gitbook install`
- `gitbook serve`

See more on installation, setup, and building here: <https://toolchain.gitbook.com/setup.html>
